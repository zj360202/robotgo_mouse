# robotgo_mouse
robotgo_mouse只包含鼠标模拟操作的robotgo简化版，

以下是我个人理解与测试，错误的地方请原谅；
  1. keyboard感觉好像少几个按键( "-", "=", "[", "]", "\", ";", "'", ",", ".", "/")；
  2. 截屏功能时间开销又太大；(测试截屏到保存一张png开销在500ms)

所以就单独提取了mouse一块的功能

从完整引用robotgo项目生产exe 10MB到现在3.5MB

[robotgo](https://github.com/go-vgo/robotgo) 原项目
